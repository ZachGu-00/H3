# Data Directory

The demo uses CSV edge lists stored directly in this folder.

Expected files for the bundled demo:

```text
data/
|-- 2014AK_30.csv
|-- 2014AK_90.csv
`-- 2014AK_180.csv
```

Minimum files:

- `2014AK_30.csv` for Task A
- `2014AK_90.csv` for Task B

Run the demo from the repository root:

```bash
python run_demo.py
```
